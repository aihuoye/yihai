# 管理后台快速开始

## 第一步：启动服务

确保后端服务已启动：

```bash
cd backend
npm start
```

服务将在 `http://localhost:4000` 运行。

## 第二步：访问登录页面

在浏览器中打开：
```
http://localhost:4000/admin/login.html
```

## 第三步：登录

输入以下信息：
- **用户名**: `admin`
- **密码**: `Aa121212`
- **验证码**: 输入图片中显示的4位验证码

点击"登录"按钮。

## 第四步：管理医生信息

登录成功后，默认进入"医生基础信息管理"页面。

### 添加第一位医生

1. 填写医生信息：
   - 姓名：张医生
   - 头衔：主任医师
   - 所在医院：XX医院
   - 科室：男科
   - 挂号费用：50.00
   - 擅长：前列腺疾病治疗
   - 简介：从事男科临床工作20年

2. 点击"保存"按钮

3. 医生信息会显示在下方的列表中

## 第五步：设置号源

1. 点击左侧菜单的"医生号源信息管理"

2. 选择刚才添加的医生

3. 设置日期范围（默认是今天到未来6天）

4. 点击"批量设置号源"按钮

5. 输入上午号源数量：`20`

6. 输入下午号源数量：`20`

7. 确认设置

8. 点击"查询号源"查看设置结果

## 第六步：测试完成

现在你已经完成了基本设置：
- ✅ 添加了医生信息
- ✅ 设置了号源

用户可以在小程序中看到这位医生，并进行预约了！

## 快捷操作

### 快速设置一周号源
```
医生：选择医生
开始日期：2025-12-04
结束日期：2025-12-10
批量设置：上午20，下午20
```

### 快速设置一个月号源
```
医生：选择医生
开始日期：2025-12-04
结束日期：2026-01-03
批量设置：上午30，下午30
```

## 常用功能速查

| 功能 | 位置 | 操作 |
|------|------|------|
| 添加医生 | 医生基础信息管理 | 填写表单 → 保存 |
| 编辑医生 | 医生基础信息管理 | 点击编辑 → 修改 → 保存 |
| 删除医生 | 医生基础信息管理 | 点击删除 → 确认 |
| 批量设置号源 | 医生号源信息管理 | 选择医生和日期 → 批量设置 |
| 查询号源 | 医生号源信息管理 | 选择医生和日期 → 查询号源 |
| 修改号源 | 医生号源信息管理 | 查询后直接修改 → 保存修改 |
| 退出登录 | 顶部右上角 | 点击退出登录 |
