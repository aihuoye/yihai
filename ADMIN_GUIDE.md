## 二、医生基础信息管理

### 功能说明
在左侧菜单点击"医生基础信息管理"进入该页面。

### 主要功能

#### 1. 添加医生
填写以下信息后点击"保存"按钮：
- 姓名（必填）
- 头衔
- 所在医院
- 科室
- 挂号费用（默认10.00元）
- 头像（支持上传图片）
- 擅长
- 简介

#### 2. 编辑医生
- 点击医生列表中的"编辑"按钮
- 修改信息后点击"保存"

#### 3. 删除医生
- 点击医生列表中的"删除"按钮
- 确认后删除

#### 4. 重置表单
点击"重置"按钮清空表单内容

## 三、医生号源信息管理

### 功能说明
在左侧菜单点击"医生号源信息管理"进入该页面。

### 主要功能

#### 1. 查询号源
1. 选择医生
2. 选择开始日期和结束日期
3. 点击"查询号源"按钮

查询结果会以表格形式展示，包括：
- 日期
- 星期
- 上午号源数量
- 下午号源数量
- 总号源数量

#### 2. 批量设置号源
1. 选择医生
2. 选择日期范围
3. 点击"批量设置号源"按钮
4. 输入上午号源数量（如：20）
5. 输入下午号源数量（如：20）
6. 确认后批量设置

**说明**：批量设置会覆盖指定日期范围内的所有号源数据。

#### 3. 单独修改号源
1. 先查询出号源列表
2. 直接在表格中修改上午或下午的号源数量
3. 点击"保存修改"按钮

**说明**：可以同时修改多个日期的号源，一次性保存。

#### 4. 号源可视化展示
- 表格清晰展示每天的号源情况
- 显示星期信息，方便识别周末
- 实时计算总号源数量

## 四、使用技巧

### 1. 快速设置一周号源
```
1. 选择医生
2. 开始日期：今天
3. 结束日期：今天+6天
4. 点击"批量设置号源"
5. 输入上午号源：20
6. 输入下午号源：20
```

### 2. 调整特定日期号源
```
1. 查询出包含该日期的号源列表
2. 直接修改对应日期的数量
3. 点击"保存修改"
```

### 3. 查看医生排班情况
```
1. 选择医生
2. 选择较长的日期范围（如一个月）
3. 查询后可以看到整月的排班情况
```

## 五、注意事项

1. **登录会话**：关闭浏览器后需要重新登录
2. **号源设置**：批量设置会覆盖原有数据，请谨慎操作
3. **日期范围**：建议一次查询不超过30天，避免数据过多
4. **删除医生**：删除医生不会删除相关的号源和预约记录
5. **修改保存**：修改号源后必须点击"保存修改"才会生效
