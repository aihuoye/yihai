# 管理后台功能截图说明

## 一、登录页面

### 访问地址
```
http://localhost:4000/admin/login.html
```

### 页面特点
- 🎨 渐变紫色背景，现代化设计
- 🔐 用户名和密码输入框
- 🖼️ 动态图形验证码
- 🔄 点击验证码可刷新
- ✨ 登录按钮悬停效果

### 登录信息
```
用户名：admin
密码：Aa121212
验证码：输入图片中的4位字符
```

### 验证码特点

- 字符随机旋转和位移
- 点击刷新

```
┌─────────────────────────────────────────────────┐
│  医生号源信息管理                                │
├─────────────────────────────────────────────────┤
│                                                 │
│  [查询条件]                                      │

│  │ 选择医生 │ 开始日期 │ 结束日期 │ [查询] │   │
│  └──────────┴──────────┴──────────┴────────┘   │
│                                                 │
│  [批量操作]                                      │
│  [批量设置号源] [保存修改]                       │

│  [号源列表表格]                                  │
│  ┌──────────┬──────┬──────────┬──────────┬────┐│
│  │ 日期     │ 星期 │ 上午号源 │ 下午号源 │总数││
│  ├──────────┼──────┼──────────┼──────────┼────┤│
│  │2025-12-04│星期三│   [20]   │   [20]   │ 40 ││

│  │2025-12-06│星期五│   [20]   │   [20]   │ 40 ││
```

### 功能说明

#### 查询条件
- 医生下拉选择框
- 日期范围选择器（开始日期 + 结束日期）
- 蓝色查询按钮

#### 批量操作
- 批量设置号源：弹窗输入上午和下午号源数量
- 保存修改：保存表格中修改的数据

#### 号源列表
- 显示日期和星期
- 上午/下午号源：可编辑的输入框
- 总数：自动计算（只读）
- 输入框样式：居中对齐，宽度60px

## 五、交互效果

### 按钮效果
- **主要按钮（蓝色）**
  - 正常：蓝色背景
  - 悬停：深蓝色 + 向上移动2px + 阴影
  - 点击：恢复原位

- **次要按钮（灰色）**
  - 正常：浅灰色背景
  - 悬停：深灰色背景

- **危险按钮（红色）**
  - 正常：红色背景
  - 悬停：深红色背景

### 菜单效果
- 悬停：浅灰色背景
- 激活：蓝色背景 + 左侧蓝色边框 + 加粗文字

### 表单效果
- 输入框聚焦：蓝色边框
- 验证码点击：刷新图片

## 六、响应式设计

### 桌面端（推荐）
- 最佳体验
- 完整功能展示
- 宽度 ≥ 1200px

### 平板端
- 自适应布局
- 宽度 768px - 1199px

### 移动端
- 基本可用
- 建议使用桌面端操作

## 七、颜色方案

- 主色：`#4f8efd` (蓝色)

- 卡片背景：`#ffffff` (白色)
- 表头背景：`#f9fafb` (浅灰)

### 文字色
- 主文字：`#333333` (深灰)

## 八、使用建议

### 最佳实践
1. 使用Chrome、Edge或Firefox浏览器
2. 屏幕分辨率 ≥ 1366x768
3. 定期保存修改
4. 批量操作前先查询确认

### 操作流程
1. 登录系统
2. 添加医生信息
3. 设置号源
4. 定期维护更新

### 注意事项
- 批量设置会覆盖原有数据
- 删除医生前请确认
- 修改号源后记得保存
- 定期退出登录保证安全
