# 管理后台功能演示脚本

## 🎬 演示准备

### 环境要求
- 后端服务已启动（`npm start`）
- 浏览器已打开
- 数据库已配置

### 演示时间
约 10-15 分钟

---

## 📝 演示脚本

### 第一部分：登录功能演示（2分钟）

#### 场景：管理员登录系统

**步骤1：访问登录页面**
```
打开浏览器，访问：http://localhost:4000/admin/login.html
```

**讲解要点**：
- 这是我们的管理后台登录页面
- 采用现代化的渐变紫色设计
- 包含用户名、密码和验证码三重验证

**步骤2：演示验证码功能**
```
点击验证码图片，观察验证码刷新
```

**讲解要点**：
- 验证码动态生成，每次都不同
- 包含干扰线和干扰点，提高安全性
- 点击即可刷新

**步骤3：演示错误处理**
```
1. 输入错误的验证码，点击登录
   - 观察错误提示
   - 验证码自动刷新

2. 输入错误的密码，点击登录
   - 观察错误提示
```

**讲解要点**：
- 系统会给出明确的错误提示
- 验证码错误后自动刷新
- 保护系统安全

**步骤4：成功登录**
```
输入正确信息：
- 用户名：admin
- 密码：Aa121212
- 验证码：输入图片中的字符
点击"登录"按钮
```

**讲解要点**：
- 登录成功后自动跳转到管理页面
- Session会话管理，保持登录状态

---

### 第二部分：界面布局介绍（1分钟）

#### 场景：介绍管理后台的整体布局

**讲解要点**：

**顶部导航栏**：
- 左侧：系统标题"医生管理后台"
- 右侧：当前登录用户名和退出登录按钮

**左侧菜单**：
- 两个主要功能模块
- 图标 + 文字，清晰直观
- 激活状态有蓝色高亮

**主内容区**：
- 根据菜单选择显示不同内容
- 白色卡片设计，清爽简洁

---

### 第三部分：医生信息管理演示（5分钟）

#### 场景：添加一位新医生

**步骤1：填写医生基本信息**
```
在表单中填写：
- 姓名：张医生
- 头衔：主任医师
- 所在医院：XX医院
- 科室：男科
- 挂号费用：50.00
```

**讲解要点**：
- 表单采用两列布局，提高填写效率
- 挂号费用支持小数，精确到分
- 所有字段都有清晰的标签

**步骤2：上传医生头像**
```
1. 点击"选择文件"按钮
2. 选择一张医生照片
3. 观察头像预览
```

**讲解要点**：
- 支持实时预览
- 圆形头像设计，美观大方
- 图片会转换为Base64存储

**步骤3：填写详细信息**
```
- 擅长：前列腺疾病、男性不育、性功能障碍
- 简介：从事男科临床工作20年，擅长各类男科疾病的诊断和治疗
```

**讲解要点**：
- 多行文本框，方便输入长文本
- 这些信息会在小程序中展示给用户

**步骤4：保存医生信息**
```
点击"保存"按钮
```

**讲解要点**：
- 保存成功后显示提示
- 表单自动清空，可以继续添加
- 医生列表实时更新

**步骤5：演示编辑功能**
```
1. 在医生列表中找到刚才添加的医生
2. 点击"编辑"按钮
3. 修改挂号费用为 60.00
4. 点击"保存"
```

**讲解要点**：
- 点击编辑后，表单自动填充医生信息
- 可以修改任何字段
- 保存后列表立即更新

**步骤6：演示重置功能**
```
1. 在表单中填写一些内容
2. 点击"重置"按钮
3. 观察表单清空
```

**讲解要点**：
- 重置按钮可以快速清空表单
- 方便重新填写或取消编辑

### 第四部分：号源管理演示（5分钟）

#### 场景：为医生设置号源

**步骤1：切换到号源管理页面**
```
点击左侧菜单的"医生号源信息管理"
```

**讲解要点**：
- 菜单切换流畅
- 页面内容立即更新
- 激活状态清晰显示

**步骤2：批量设置号源**
```
1. 在"选择医生"下拉框中选择"张医生"
2. 开始日期：选择今天
3. 结束日期：选择今天+6天（一周）
4. 点击"批量设置号源"按钮
5. 输入上午号源：20
6. 输入下午号源：20
7. 确认
```

**讲解要点**：
- 批量设置可以快速配置多天的号源
- 支持自定义日期范围
- 上午和下午可以设置不同的数量
- 适合快速设置常规排班

**步骤3：查询号源**
```
点击"查询号源"按钮
```

**讲解要点**：
- 查询结果以表格形式展示
- 包含日期、星期、上午号源、下午号源、总数
- 星期信息方便识别周末
- 总数自动计算

**步骤4：单独修改号源**
```
1. 在表格中找到某个日期（如周六）
2. 将上午号源改为 30
3. 将下午号源改为 30
4. 点击"保存修改"按钮
```

**讲解要点**：
- 可以针对特定日期调整号源
- 支持同时修改多个日期
- 适合处理特殊情况（如节假日）
- 保存后立即生效

**步骤5：演示不同日期范围查询**
```
1. 修改日期范围为未来一个月
2. 点击"查询号源"
3. 观察更长时间范围的号源情况
```

**讲解要点**：
- 支持灵活的日期范围查询
- 可以查看长期的排班情况
- 方便提前规划

#### 场景：演示其他辅助功能

- 退出登录需要确认


**讲解要点**：
